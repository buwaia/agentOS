# Rule R001: 解答必须包含直觉性描述

## 追溯
- **Principle**: P1（洞见优先于推导）
- **Evidence**: 2026-06-27，Claude 用纯公式推导琴生不等式，用户判定为"AI幻觉"；改用半圆几何类比后用户认可

## 适用范围
- 所有数学证明题
- 错题本中的 AI 解析输出

## 判定标准
解答必须满足以下至少一项：
- 包含几何图示（ASCII 图、坐标描述、图形类比）
- 包含自然语言类比（"就像..."、"想象..."、"本质是..."）
- 包含直觉性结论先于推导（先说结论，再验证）

**不满足**：纯符号推导链，无任何图示或类比

## 违规示例
```
f''(x) = -1/x² < 0，所以 f(x) 是凹函数，
由 Jensen 不等式得 f((a+b)/2) ≥ (f(a)+f(b))/2 ...
```
（机械，无洞见）

## 满足示例
```
在直径为 a+b 的半圆里，垂线长度 = √(ab)，
而垂线不超过半径 (a+b)/2，几何事实直接给出结论。
```
（一个半圆，真相自现）
