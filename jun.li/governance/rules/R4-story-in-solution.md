# Rule R4: 解析必须包含故事或场景描述

## 追溯
- **Principle**: P2（AI 输出质量是产品底线）
- **Evidence**: 2026-06-27，学生反馈数学解题步骤全是公式看不懂，希望能像高斯一样用故事讲清楚

## 判定标准
解析里有故事或场景描述即合格，纯公式连续超过3行不合格。

## 违规示例
```
设 a, b > 0，由 AM-GM 不等式：
a + b ≥ 2√(ab)
∴ (a+b)/2 ≥ √(ab)
两边平方得 (a+b)²/4 ≥ ab
展开左边 (a²+2ab+b²)/4 ≥ ab
```
（连续5行纯公式，无任何叙事描述）

## 满足示例
想象你有两根绳子，一根长 a，一根长 b。把它们拼成一个矩形，面积是 ab。
但如果你把同样长度的绳子围成正方形，面积会更大——这就是 AM-GM 的直觉：
均值总是"更方"，所以更大。
