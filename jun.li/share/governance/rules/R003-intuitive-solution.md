# Rule R003: 解答必须通俗易懂

## 追溯
- **Principle**: P2（AI 输出质量是产品底线）
- **Evidence**: 2026-06-27，用户反馈网上找到的单调性证明看不懂；Sara 给出的符号推导同样缺乏通俗感；用户要求参考拉马努金、高斯风格——用生活化类比让人一眼看懂

## 判定标准
解答包含通俗类比或生活化描述即合格，纯符号推导链不合格。

## 违规示例
```
f(x₁) - f(x₂) = (x₁-x₂)(x₁²+x₁x₂+x₂²+2)
因为 x₁ < x₂，所以 (x₁-x₂) < 0，括号内 > 0，故差 < 0，得证。
```
（纯符号，学生看不懂为什么这样做）

## 满足示例
```
想象一条一直往上爬的山路——不管从哪里出发，走得越远就越高。
x³+2x 就是这样的函数：x 越大，值越大，永远不会"折回去"。
证明只是把这个直觉用数字验证一遍。
```
（先给画面，再给证明）
